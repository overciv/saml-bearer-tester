Resource Owner Password (ROPC)
⚠️ Legacy flow — use only when no other flow works. The application handles the user's credentials directly. Only suitable for highly trusted first-party apps. OAuth 2.1 deprecates this flow.
Setup: Enable Resource Owner Password grant type on the Okta app (Apps → your app → General → Grant types → Advanced), then add it to the authorization server policy rule.
